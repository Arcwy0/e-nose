# data/ — Training data + generated plots

## What's tracked

- `enose_points_1906.csv` — small reference dataset kept in git as a smoke
  test / example for `/smell/learn_from_csv`.

## What's ignored

- `data/*.csv` other than the reference file above — put your own
  training logs here; they stay local unless you explicitly `git add -f`.
- `data/plots/` — regenerated by `/smell/visualize_data` on demand.
- `database_robodog*.csv` at the repo root — accumulated training log
  written by `save_training_samples`.

To share a large CSV with a collaborator, either:

- send it out-of-band, or
- force-add it (`git add -f data/my_big_dataset.csv`) — Git LFS takes
  over via the rule in `.gitattributes`.
